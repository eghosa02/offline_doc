\section{Ambiente Operativo}

\subsection{Ambiente di Sviluppo}

- Sistema operativo: Windows 10/11, Ubuntu 20.04+, macOS 10.15+.
- Python 3.8+.
- Editor: VS Code con estensioni LaTeX.
- Version control: Git.

\subsection{Ambiente di Distribuzione}

- Applicazione desktop standalone.
- Installazione tramite installer o package manager.
- Nessun server richiesto.

\subsection{Ambiente di Test}

- Ambiente isolato per testing unitario e integrazione.
- Strumenti: pytest, Selenium per UI testing.
