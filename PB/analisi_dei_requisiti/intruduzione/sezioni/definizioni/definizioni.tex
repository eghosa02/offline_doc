\section{Definizioni}

\subsection{Termini Chiave}

- **Sistema**: L'applicazione MiroFish Offline Doc.
- **Utente**: Persona che utilizza il sistema per gestire documentazione.
- **Documento**: File contenente informazioni tecniche o manuali.
- **Requisito**: Descrizione di una funzionalità o vincolo del sistema.

\subsection{Acronimi}

- SRS: Software Requirements Specification
- GUI: Graphical User Interface
