\section{Casi d'Uso}

\subsection{Caso d'Uso Principale: Creazione Documento}

\textbf{Attore:} Sviluppatore

\textbf{Descrizione:} L'utente crea un nuovo documento nel progetto.

\textbf{Precondizioni:} Progetto aperto.

\textbf{Postcondizioni:} Documento creato e pronto per editing.

\textbf{Flusso Principale:}
1. Selezionare "Nuovo Documento" dal menu.
2. Scegliere tipo (LaTeX/Markdown).
3. Inserire nome e posizione.
4. Il sistema crea il file e apre l'editor.

\subsection{Caso d'Uso: Esportazione PDF}

\textbf{Attore:} Utente

\textbf{Descrizione:} L'utente esporta un documento in PDF.

\textbf{Precondizioni:} Documento valido aperto.

\textbf{Postcondizioni:} File PDF generato.

\textbf{Flusso Principale:}
1. Selezionare "Esporta PDF".
2. Scegliere posizione di salvataggio.
3. Il sistema compila e salva il PDF.
