\section{Requisiti Funzionali}

\subsection{Gestione Progetti}

- RF1: Il sistema deve permettere la creazione di nuovi progetti.
- RF2: Il sistema deve supportare l'importazione di progetti esistenti.

\subsection{Editing Documenti}

- RF3: Il sistema deve fornire un editor integrato per LaTeX.
- RF4: Il sistema deve supportare la modifica di file Markdown.

\subsection{Esportazione}

- RF5: Il sistema deve esportare documenti in PDF.
- RF6: Il sistema deve supportare esportazione in HTML.
