\section{Introduzione}

MiroFish-Offline è un'applicazione completa basata su architettura moderna che combina:
\begin{itemize}
    \item Backend Flask con API REST
    \item Database Neo4j per grafi relazionali
    \item Ollama per modelli LLM locali
    \item Frontend Vue.js
\end{itemize}

Questo manuale descrive come avviare l'intera applicazione utilizzando Docker, che semplifica notevolmente il deployment e garantisce coerenza tra ambienti.
