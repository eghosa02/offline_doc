\section{Prospettiva del Sistema}

Il sistema MiroFish Offline Doc è un'applicazione desktop per la gestione di documentazione tecnica. Fornisce un'interfaccia utente intuitiva per la creazione e modifica di documenti in formato LaTeX e Markdown, con possibilità di esportazione in PDF.

\subsection{Contesto del Sistema}

- Interfaccia con il file system locale.
- Supporto per progetti strutturati in cartelle.
- Integrazione con editor di testo esterni.
