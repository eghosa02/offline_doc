\section{Prerequisiti}

Prima di iniziare, assicurati di avere installato:

\subsection{Requisiti di Sistema}

\begin{itemize}
    \item \textbf{Docker Desktop} (versione 4.0 o superiore)
    \item \textbf{Spazio su disco}: almeno 50 GB disponibili
    \item \textbf{RAM}: minimo 8 GB (consigliati 16 GB)
    \item \textbf{GPU NVIDIA} (opzionale, per accelerazione Ollama)
\end{itemize}

\subsection{Verificare l'Installazione di Docker}

Apri il terminale PowerShell e verifica che Docker sia installato:

\begin{lstlisting}
docker --version
docker-compose --version
\end{lstlisting}

Dovresti vedere versioni simili a:
\begin{lstlisting}
Docker version 27.0.0, build ...
Docker Compose version 2.x.x, ...
\end{lstlisting}
